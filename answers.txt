Part 1 1.When I type a website address like google.com and press Enter, a request is being sent. The Domain Name System (DNS) converts google.com into its particular IP address because computers understand IP addresses and not names. The browser then sends the request to that IP address, where it reaches the load balancer. The load balancer distributes the request to one of the available web servers. The web server processes the request and sends the webpage back to my browser, where the Google landing page is displayed.
